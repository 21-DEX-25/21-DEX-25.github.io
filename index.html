Le projet central de ma première année de BTS : une infrastructure complète déployée sur VMware Workstation.

🗂️
AD DSDNS / DHCP
Contrôleurs de domaine devida.local

Déploiement d'un domaine Active Directory sur Windows Server 2022, avec un second contrôleur de domaine pour la redondance DNS/DHCP.

📦
TrueNASPartage

Mise en place d'un serveur TrueNAS avec un dossier partagé accessible à tous les utilisateurs du domaine DEVIDA.
